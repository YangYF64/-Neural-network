# C++ BP神经网络 (C++ BP Neural Network)

## 📂 项目结构

```
.
├── include/          # 存放头文件 (bpNet.h)
├── src/              # 存放源文件 (bpNet.cpp, main.cpp)
├── output/           # 存放最终生成的可执行文件
├── .vscode/          # Visual Studio Code 配置文件 (tasks.json, launch.json)
├── data.txt          # 默认的训练数据文件
└── README.md         # 项目说明文件
```
## 🚀 如何构建与运行
### 环境要求
能跑C++就行

## 📝 如何使用
直接F5就行，程序会提示输入**目标误差**和**最大训练轮数**。
训练结束后，程序会打印详细的总结报告，并展示网络在训练集上的最终预测效果
